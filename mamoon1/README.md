# Mamoon 專案網站

Mamoon 專案的官方網站。這是一個單頁式網站，展示專案的主要特點、團隊成員和聯絡方式。

## 特點

- 單頁式設計
- 黑白風格搭配視差滾動效果
- 移動裝置適配
- 動畫效果

## 技術使用

- HTML5
- CSS3
- JavaScript (原生)

## 圖片來源
